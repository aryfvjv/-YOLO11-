# -YOLO11-
YOLO11的一个场景应用的尝试
1.（适用于初学者）物体检测定义为物体定位+物体分类。YOLO11 通过将输入图像传入 CNN 来提取特征，从而执行对象检测。然后，网络预测这些网格内对象的边界框和类别概率。为了处理多尺度检测，使用层来确保检测到各种大小的对象。然后使用非最大抑制 (NMS)细化这些预测以过滤掉重复或低置信度的框，从而实现更准确的对象检测。YOLO11 在 MS-COCO 数据集上进行对象检测训练，其中包括 80 个预训练类。

2.为什么选择那里yolo11作为检测的工具？
因为在Backbone:\
（1）	YOLO11引入了C3k2, 该块是CSP Bottleneck的自定义实现，它使用两个卷积\
CSP 网络分割特征图，并通过瓶颈层处理一部分，同时将另一部分与瓶颈的输出合并。这减少了计算量并改善了特征表示。\
（2）	C3k2 块还使用较小的内核大小（由 k2 表示），使其速度更快同时保持性能。\
（3）	C2PSA （跨阶段局部空间注意）模块增强了特征图中的空间注意，提高了模型对图像重要部分的关注。通过空间池化特征，模型能够更有效地关注特定感兴趣区域。\
（4）	YOLO11 将颈部的 C2f 块替换为C3k2块。如前所述，C3k2 是一个更快、更高效的块。\
（5）	C3k2 块：与颈部类似，YOLO11 替换了头部的C2f块.。\
注：C3k2是CSP bottleneck的一个更快、更高效的变体。它使用两个卷积而不是一个大卷积，从而加快了特征提取速度。\

3.（YOLO11 是基于 YOLOv8 代码库构建的）代码库分为多个模块，这些模块定义了 YOLO11 模型中使用的各种神经网络组件。 这些组件被组织到 nn/modules/ 目录中的不同文件中：
block.py：定义模型中使用的各种构建块（模块），例如瓶颈、CSP 模块和注意力机制。\
conv.py：包含卷积模块，包括标准卷积、深度卷积和其他变体。\
head.py：实现负责产生最终预测（例如，边界框、类概率）的模型头部。\
transformer.py：包括基于变压器的模块，用于注意机制和高级特征提取。\
utils.py：提供跨模块使用的实用函数和辅助类。\
nn/tasks.py文件定义了不同的特定任务模型（例如，检测、分割、分类），这些模型将这些模块组合起来形成完整的架构。\

4.数据处理
YOLO11要求数据必须满足Ultralytics YOLO 格式。它允许您定义数据集根目录、训练/验证/测试图像目录的相对路径或 *.txt 文件（包含图像路径）和类名字典。该格式的标签应导出为YOLO 格式，并带有一个 *.txt 文件。如果图像中没有对象，则没有 *.txt 文件是必需的。文件 *.txt 文件格式，每个对象在 class x_center y_center width height 格式。方框坐标必须使用 归一化 xywh 格式（从 0 到 1）。如果您的方框以像素为单位，则应将其除以 x_center 和 width 图像宽度，以及 y_center 和 height 由图像高度决定。类号应为零索引（从 0 开始）。\
数据集来源：飞桨\
数据集处理：labelimg（labes代表每个检测到的目标的类别和边界框信息，每个目标都是一个矩形边界框和一个类别标签）\

5.训练结果分析
Weights文件夹，两个文件，分别是best.pt,last.pt文件，分别存储了训练过程中的损失结果和模型训练的最后一次结果保存的模型。Args.yaml文件主要保存了训练时的指定参数。
关键需要明确的概念：\
[1]	精确度（PRECISION）：预测为正的样本中有多少个是正确的。Precision=TP/(TP+	FP)\
[2]	召回率（RECALL）：真实为正的样本有多少被正确预测为正。Recall=TP/(TP+FN)\
[3]	F1值（F1-SCORE）：综合考虑精确度和召回率的指标。F1=2*(Presision*Recall)/(Precision+Recall)\
[4]	准确度（ACCURACY）:所有样本中模型正确预测的比例。Accuracy=(TP+TN)/(TP+TN+FP+FN)\
[5]	平均精确度（AVERAFE PRECISION）AP：用于计算不同类别的精度，对于二分类问题，AP=精确度对于多分类问题则为MAP：\            
[6]	FPS：每秒帧率。用于评估模型在给定硬件的处理速度。\
[7]	IoU:全程为intersection over Union表示交并比。在目标检测过程中衡量模型生成的候选框与原标记框之间的重叠程度\
混淆矩阵：用于评估分类模型性能的表格形式，以实际类别（真实值）和模型预测类别为结果（二分类问题）\
true代表我真实的标签是焊缝，但是我预测出来的是背景，（后面这个是归一化）\
plot_metrics 方法旨在从指定的 CSV 文件中提取训练和验证的指标数据，并使用 plot_results 函数生成图形，该result.csv记录了我们训练过程中的参数信息，包括损失和学习率。\
